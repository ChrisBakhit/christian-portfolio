# Christian Bakhit — Portfolio

My personal software engineering portfolio, built as an interactive VS Code-inspired workspace with a cinematic space theme.

**Live site:** [christian-portfolio-chrisbjhus-projects.vercel.app](https://christian-portfolio-chrisbjhus-projects.vercel.app)

## About

The portfolio presents my experience across full-stack product development, cloud infrastructure, applied AI, patent intelligence, research software, and community projects. Each section is represented as a real editor file—including `profile.md`, `experience.js`, `projects.json`, `skills.json`, and `education.md`.

## Features

- Interactive VS Code-style navigation, Explorer, menus, command palette, and terminal
- Searchable experience, projects, skills, and education
- Live project previews linking directly to deployed products
- Animated shooting stars, constellations, orbital systems, and black-hole interactions
- Scroll-aware file highlighting and progress indicators
- Keyboard shortcuts and hidden terminal commands
- Responsive layouts for desktop, tablet, and mobile
- Downloadable resume

## Technology

- Next.js 16
- React 19
- TypeScript
- CSS animations and responsive design
- Vercel deployment with GitHub automatic deployments

## Run Locally

Requires Node.js 22.13 or newer.

```bash
git clone https://github.com/ChrisBakhit/christian-portfolio.git
cd christian-portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Create a production build with:

```bash
npm run build
```
